Interactive music synchronization

Drag and drop audio files (.wav, .mp3, .ogg, .flac), MusicXML scores (.xml, .musicxml), and/or MIDI files (.mid, .midi) into the player below. Select one file as the reference timeline, then click Synchronize to synchronize all files using your selected warping-path features and algorithm. Alignment is computed entirely in your browser using synctoolbox via Pyodide.
